WiFi, Bluetooth and 802.15.4 
A single module provides all 3 radios - ublox MAYA W2. This module is based on the NXP IW612 chipset.
https://www.nxp.jp/products/wireless-connectivity/wi-fi-plus-bluetooth-plus-802-15-4/2-4-5-ghz-dual-band-1x1-wi-fi-6-802-11ax-plus-bluetooth-5-4-plus-802-15-4-tri-radio-solution:IW612
In case it offers some insight, the i.MX93 Quick Start Evaluation Kit uses an M.2 module for wireless comms so must have the driver installed. This uses the Murata 2EL module which is IW612 based - LBES5PL2EL-923 
https://www.nxp.jp/design/design-center/software/development-software/i-mx-93-quick-start-evaluation-kit:IMX93QSB


Device Serial Numbers


Device Pinout:
Powerdown - This functionality is controlled by an external low power micro MCXC143VFM. This is to allow the WiFi module to remain powered while the i.MX93 is not powered. By default, pullup resistors force the WiFi module into the ON state as the MCXC143VFM is not yet programmed.
